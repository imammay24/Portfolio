About Me

Description

 Saya Imam Maulana, seorang Mahasiswa STMIK IKMI Cirebon Angkatan Tahun 2020. Saya berasal dari Majalengka dan sekarang berdomisili di Cirebon. Lingkup keterampilan dan minat saya dominan di bidang IT khususnya Web Development.

 Ketertarikan saya terhadap Web Development sangat tinggi baik itu Front-End maupun Back-End. Fokus saya sekarang ada di Front-End Web Development, tapi itu tidak menutup kemungkinan saya untuk menjamah seputar Back-End Web Development.

 Walaupun saya tidak memiliki pengalaman kerja tetapi saya pernah melakukan Praktek Kerja Industri serta menjadi seorang Freelancer di suatu instansi. Saya memiliki keingintahuan yang tinggi sehingga saya selalu ingin mempelajari hal-hal yang baru. Selain itu, saya juga memiliki kecermatan serta ketelitian yang lumayan tinggi ketika saya akan berlayar di dunia IT.
